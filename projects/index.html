    🛠️ Aplikasi & Tools yang Sering Saya Gunakan

    Untuk mendukung proyek-proyek dan eksplorasi saya, berikut adalah beberapa aplikasi dan tools yang sering saya manfaatkan:

    • Cisco Packet Tracer, GNS3, PNETLAB: Untuk simulasi dan desain jaringan yang realistis.
    • Wireshark, Nmap, Metasploit: Untuk analisis jaringan, port scanning, dan penetration testing dasar.
    • Debian, Ubuntu, CentOS: Sistem operasi pilihan untuk server, pengembangan, dan lab virtual.
    • Docker, Portainer: Untuk containerization, orkestrasi, dan manajemen aplikasi terisolasi.
    • VS Code, Git, GitHub: Lingkungan pengembangan utama dan sistem version control terintegrasi.
    • Proxmox VE, VirtualBox: Untuk virtualisasi server, manajemen VM, dan pembangunan lab virtual.
    • n8n, Ansible: Untuk otomatisasi alur kerja tanpa kode dan manajemen konfigurasi server.
